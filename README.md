# 🏠 Real Estate Buyer Segmentation & Market Intelligence Analytics

## Project Overview

This project focuses on analyzing real estate customer behavior and transaction patterns using Machine Learning and Business Intelligence techniques.

The solution combines data cleaning, feature engineering, buyer segmentation, transaction segmentation, Power BI dashboards, and an interactive Streamlit application to generate actionable business insights for real estate organizations.

The objective is to identify distinct buyer groups, understand investment behavior, analyze transaction trends, and support data-driven decision-making.

---

## Business Problem

Real estate companies often manage thousands of transactions and customers with different purchasing behaviors.

Without proper segmentation, it becomes difficult to:

* Identify high-value investors
* Understand buyer preferences
* Analyze transaction patterns
* Improve marketing strategies
* Enhance customer experience
* Support business decision-making

This project addresses these challenges using machine learning-based segmentation and interactive dashboards.

7 Machine Learning Based Buyer Segments were identified using K-Means Clustering.

* Conservative Investors
* Experienced Buyers
* High-Value Investors
* Mid-Value Investors
* Premium Investors
* Stable Investors
* Standard Property Buyers
